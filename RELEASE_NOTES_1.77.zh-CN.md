# Litrix 1.77 发布说明

发布日期：2026-04-01

## 重点更新

1. 优化了整体界面设计与信息层次，主界面更适合长时间阅读与矩阵整理。
2. 高级检索支持将文库范围锁定到 `Library`、`Collection`、`Tag` 等分类，并支持从检索结果直接创建新的 `Collection`。
3. 新增 `.litrix` 归档导入导出，可对 `settings`、`papers`、`notes` 等核心数据进行备份、迁移与分享。
4. 文献矩阵支持隐藏列、左右移动列与重排列顺序。
5. 文献矩阵列表支持直接编辑，减少切换成本。
6. 搜索支持文内引用格式查询。
7. 新增 `Zombie Collection`，帮助识别长期未使用文献并回看历史收藏决策。
8. 支持为 `Tag` 自定义快捷键。
9. 工具栏支持 API 连接状态检测。
10. 元数据刷新支持按字段拆分 prompt，仅发送当前目标元数据所需内容，以降低 token 消耗。
11. 工具栏支持显示元数据提取进度。
12. 新增高级筛选，交互风格参考 Numbers。
13. 优化右侧栏排版，新增顶部固定标题，提升信息密度。
14. 完善设置界面，支持中文等基础能力。
15. 内置 Litrix MCP，可让 AI 读取 PDF 全文、修改元数据、写入 note、管理标签与集合。
16. 存储结构进一步收敛：note 固定为 `note.txt`，图片固定落在 `images/`，更利于稳定读写与后续扩展。
17. 多处性能体验已优化，包括导入导出排队、索引逻辑与筛选切换流畅度。

## 安装包

- 文件：`Litrix-1.77.dmg`
- 架构：Universal（`arm64` + `x86_64`）
