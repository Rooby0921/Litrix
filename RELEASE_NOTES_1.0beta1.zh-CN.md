# Litrix 1.0beta1 发布说明

发布日期：2026-03-26

## 重点更新（P1-P15）

1. （P1）优化了整体界面设计，统一视觉风格与信息层次。
2. （P2）高级检索支持将文库范围锁定到 `Library`、`Collection`、`Tag` 等分类，并支持从检索结果直接创建新的 `Collection`。
3. （P3）新增自定义文件系统 `.litrix`（你提到的 `.litirx`），可对 `settings`、`papers`、`note` 等核心数据进行备份，也支持从他人的矩阵库导入。
4. （P4）支持隐藏列，并支持列的左右移动与重排。
5. （P5）支持在文献矩阵列表中直接编辑内容。
6. （P6）搜索支持文内引用格式搜索。
7. （P7）新增 `Zombie Collection`（僵尸文献）功能，帮助识别长期未使用文献；可用于清理空间，也可反思收藏决策，减少“收藏迷航”。
8. （P8）支持为 `Tag` 自定义快捷键。
9. （P9）支持在工具栏检测 API 连接状态。
10. （P10-P11）支持自定义元数据更新类型，并对提示词进行拆分：仅发送当前目标元数据对应的 prompt，而非全量发送，以节约 token。
11. （P12）工具栏支持显示元数据提取进度。
12. （P13）支持高级筛选，交互风格参考 Numbers。
13. （P14）优化右侧栏排版，新增顶部固定标题，减少无效版面占用并提升信息密度。
14. （P15）优化设置界面，新增“中文”、深色模式等基础能力。
15. 优化多处性能体验：例如导入导出增加排队等待机制，切换 `Collection` 与 `Tag` 更流畅，索引逻辑优化后整体卡顿明显减少。

## 其他更新

- 更新了应用图标。虽然仍在继续打磨，但相比第一版更耐看，黑色方案更有质感。
